Inställningar

1 · intervals.icu

  1. Settings → Developer → kopiera API key.
  2. Athlete ID är i-numret i din profil-URL.
  3. Sätt zoner under Settings: max-HR, LTHR och FTP — annars blir zonberäkningen fel.

2 · Google Sheet (två flikar)

Flik Plan, exakt dessa kolumnrubriker:

typ namn start slut mal_lag anteckning

typ = fas, race eller vila (återhämtningsvecka). Datum som ÅÅÅÅ-MM-DD. mal_lag = mål-% lågintensivt (t.ex. 75).

Flik Delmal:

gren beskrivning plandatum faktiskt status varde anteckning

gren = sim/cykel/löp/brick/styrka · status = klart/pågår/skjutet/risk (tomt = planerat) · varde = valfri siffra, t.ex. längsta kontinuerliga simdistans i meter.

3 · Publicera flikarna som CSV

  1. I Sheet: Arkiv → Dela → Publicera på webben.
  2. Välj fliken Plan, format CSV, publicera, kopiera länken hit.
  3. Upprepa för fliken Delmal.
