2 · Google Sheet (två flikar)

Flik Plan, exakt dessa kolumnrubriker:

typ namn start slut mal_lag kategori anteckning

typ = fas, race eller vila (återhämtningsvecka). Datum som ÅÅÅÅ-MM-DD. mal_lag = mål-% lågintensivt (t.ex. 75). kategori = valfri etikett ovanför tävlingsnamnet på race-rader (t.ex. Marathon, Olympisk distans, Gravel 120 km).

Flik Delmal:

gren beskrivning plandatum faktiskt status varde anteckning

gren = sim/cykel/löp/brick/styrka · status = klart/pågår/skjutet/risk (tomt = planerat) · varde = valfri siffra, t.ex. längsta kontinuerliga simdistans i meter.

Flik Race (valfri — visar tävlingsdagskortet)

race gren mal split strategi distans

En rad per gren: gren = Sim/T1/Cykel/T2/Löp/Total. mal = mål (t.ex. 2:05/100m, 225 W). split = beräknad tid. strategi = en rad coach-instruktion. race = tävlingens namn (matchar Plan); lämna tomt så gäller raden närmaste tävling. distans = valfri, för beredskapskortet vid udda lopp (sim i meter, cykel/löp i km) — vanliga kategorier behöver den inte.
